# WinFormsCargoApp (3)
# Cargo Company — Desktop App (C# + SQLite)

Десктопное приложение для учёта клиентов, тарифов и заказов на грузоперевозки.  
Выполнено в рамках лабораторной работы №5 по дисциплине «Программирование».

##  Основные функции
-  Работа с тремя сущностями: **Клиенты**, **Тарифы**, **Заказы**
-  Поддержка **постоянных клиентов** (скидка 10%)
-  Автоматический подбор тарифа по весу груза
-  **CRUD-операции**: добавление, редактирование, удаление
-  Валидация ввода и защита целостности данных (нельзя удалить, если используется в заказах)
-  Сортировка данных по клику на заголовок столбца
-  Импорт и экспорт всей базы в/из **JSON**
-  Автоматическое создание БД при первом запуске

##  Технологии
- Язык: **C#** (.NET 6+)
- UI: **Windows Forms**
- База данных: **SQLite** (файл `cargo.db`)
- Используемые компоненты:
  - `DataGridView` — отображение таблиц
  - `OpenFileDialog` / `SaveFileDialog` — работа с файлами
  - `BindingList<T>` — автоматическое обновление интерфейса

##  Структура проекта
- `Logic/` — бизнес-логика (`Client`, `Tariff`, `Order`)
- `Services/` — работа с БД (`ClientRepository`, `OrderRepository`, `Database`, `FileService`)
- `Models/` — DTO для сериализации (`DataModel`)
- Главные формы: `MainForm`, `ClientForm`, `OrderForm`, `TariffForm`

##  Как запустить
1. Откройте `WinFormsCargoApp.sln` в **Visual Studio 2022** (или новее)
2. Соберите проект (Ctrl+Shift+B)
3. Запустите (F5)
4. При первом запуске автоматически создастся файл `cargo.db` с демо-данными

>  Для работы требуется установленный **.NET 6+ SDK**.
